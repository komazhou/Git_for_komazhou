-------------------------------------------------------------------
挂载符号在当前用户输入方式文件下输入代码：

patch:
  "punctuator/import_preset": 符号文件名（没有后缀）
  "recognizer/patterns/punct": "^/(按键范围)$"

如五笔拼音：
文件：wubi_pinyin.custom.yaml

patch:
  "punctuator/import_preset": myfuhao
  "recognizer/patterns/punct": "^/([A-Z]+|[a-z]+|[0-9]+|[10])$"

横排选词：

　　在【小狼毫】用户文件夹，修改weasel.custom.yaml文件，在patch:后加入 "style/horizontal": true
竖排选词：                                                             "style/color_scheme": aqua

　　选词数修改：

　　在【小狼毫】用户文件夹，修改default.custom.yaml文件，在patch:后加入"menu/page_size": 9

　　修改F4快捷键：

　　在【小狼毫】用户文件夹，修改default.custom.yaml文件，在patch:后加入"switcher/hotkeys":    -"Control+grave"

　　修改Shift切换键：

　　在【小狼毫】用户文件夹，修改default.custom.yaml文件，在patch:后加入  "ascii_composer/switch_key":

　　Shift_L: commit_code
　　Shift_R: commit_code

1、调整字号、字体
# weasel.custom.yaml

patch:
  "style/font_face": "明兰"  # 字w名Q，挠事本等的系y字wυ框Y能看到
  "style/font_point": 14     # 字，只J底值模不J「五」、「小五」@拥

2、调整固定词序
# wubi86.custom.yaml  以五笔为例子
patch:
  translator/enable_user_dict: false


3、横排与竖排显示、显示托盘图标
# weasel.custom.yaml
patch:
  style/horizontal: true      # 候xM排
  style/inline_preedit: true  # 惹毒a（H支持TSF）
  style/display_tray_icon: true  # @示托PD


4、导入自定义词库
:se fenc=utf8 nobomb ff=unix    词库的编码格式要求  


5、候选字数
#  default.custom.yaml 
"menu/page_size": 5  # 定制每页候选字数， 这里是对整个输入法，可以单独针对不同的输入方案

6、显示托盘图标
# weasel.custom.yaml
patch:
  style/horizontal: true      # 候xM排
  style/inline_preedit: true  # 惹毒a（H支持TSF）
  style/display_tray_icon: true  # @示托PD

7、用户数据同步选项
# installation.yaml
sync_dir: 'D:\Dropbox\RimeSync'
# 又或者是
installation_id: 'fred-macbook'


8、换页键位
## 可支持全局与输入方案
  "key_binder/bindings":
    - { when: paging, accept: bracketleft, send: Page_Up }
    - { when: has_menu, accept: bracketright, send: Page_Down }

9、备份输入快照与导入、导出码表
可结合深蓝词库来做转换

10、地球拼音更改声调设置


11、关闭码表输入连打
# cangjie5.custom.yaml  只对五笔、仓颉有效
patch:
  translator/enable_sentence: false


※2015-07-31 更新内容

间断有一年多没折腾rime输入法了，最近在安装xubuntu时，又想起这个优秀的输入法框架来着，读回此前潦草记录的小狼毫设置，惭愧惭愧。

重拾rime，简单说两句吧。

「关于五笔」
上述写的11个自定义设定点，在最近的版本已经显得不那么重要了，因为默认的设定已经非常棒了。
本人五笔狗，喜欢竖排侯选字，喜欢设定侯选字的大小，这些均很方便的能在配置文件中修改而得。

「关于 中州韵」
linux下安装使用rime，根据官方指导，能够轻易上手安装。目前主要借助fitcx框架和ibus框架，两者均可，看个人口味，fitcx的皮肤较美观，ibus挫爆了。框架内变换输入法快捷键为F4；至于词库是否要搞成搜狗，五笔狗觉得无必要……五笔狗甚至不用考虑什么词频……哈哈。

「关于 鼠须管」
mac下刚开始使用QQ五笔，但在safari地址栏中输入经常卡死，后来换成清歌五笔，倒是挺爽快的，可对比rime来讲，后者比较有熟悉感，更适合个人的使用习惯，如使用z为临时拼音。另外，清歌的标点符号输入做得不好，如输入个￥都无从下手。

mac下装完rime，先在default.yaml中修改输入法选项，如本人只留五笔和地球拼音
schema_list:
  - schema: wubi86
  - schema: terra_pinyin
接着在squirrel.yaml中修改
color_scheme: aqua   候选界面主题
horizontal: true     水平显示
  font_face: 'Lucida Grande'  字体
  font_point: 19    字号
切换输入法快捷键是control+`

「关于 小狼毫」
安装过程中，会让您选择主题、输入法类型等等，是三者中最为人性化的；对于五笔狗而言，装完直接用，哈哈。输入法切换快捷键依旧是F4。

※尾巴
鼠须管、小狼毫中均更新了自动造词功能，五笔按z临时拼音功能； 对这两点大赞啊。windows中使用qq五笔，喜欢其临时拼音模式之外，还有其带英文单词输入（按分号后输入英文，能联想提醒单词），快捷输入当前日期和时间（按分号后输入date，则显示多格式当天日期；输入time则为多格式当前时间），可惜小狼毫现在不支持，鉴于保留输入的流畅性，忍痛弃之。对于词库与同步功能，五笔狗表示无爱也无用。编程随想中提到，为避免xxxx，输入法尽量使用最简洁最纯净的，因此同步功能弃之。
瘟到死中，各大公司的输入法，如搜狗、渣度、QQ等等，同步功能、用户属性过分强调，有点喧宾夺主的味道，把输入这个最纯粹的需求冷落了，啧啧。您敢想象，一个破输入法，还能提示您今天有什么新闻，这您敢想象？换回rime，输入体验比德芙如丝般顺滑更加流畅。
